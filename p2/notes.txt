Web Tarayıcısı ile Erişim İçin Hosts Dosyasını Düzenleme

Tarayıcının app1.com'un 192.168.56.110 olduğunu bilmesi gerekir. Bunun için kendi bilgisayarının (sanal makinenin değil) hosts dosyasını düzenlemelisin.

Windows'ta: 
C:\Windows\System32\drivers\etc\hosts

Linux/macOS'ta: 
/etc/hosts

Bu dosyayı yönetici/root yetkileriyle açıp şu satırları ekle:

192.168.56.110  app1.com
192.168.56.110  app2.com

Kaydettikten sonra tarayıcında:
http://app1.com adresine git -> Uygulama 1'i göreceksin.
http://app2.com adresine git -> Uygulama 2'yi göreceksin.
http://192.168.56.110 adresine git -> Varsayılan olan Uygulama 3'ü göreceksin.
